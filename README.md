# p0-w3-d1-t6 · Классификатор статус-кодов

> Тип: новый материал · Репозиторий: один отдельный

> Подключить файл, отсмотреть в браузере.

## Задача

По числовому HTTP-статусу вернуть категорию словами. Это уже маленькая логика на ветвлениях: диапазоны, `if/else if` или `switch`. Такие классификаторы постоянно встречаются в реальном коде.

```
статус-код → текст
  200..299 → 'успех'
  300..399 → 'редирект'
  400..499 → 'ошибка клиента'
  500..599 → 'ошибка сервера'
if/else if  или  switch(true) — на выбор
```

## Что нужно сделать

В `status.js` написать логику: по числу-статусу печатать категорию — 200–299 успех, 300–399 редирект, 400–499 ошибка клиента, 500–599 ошибка сервера, прочее — «неизвестный код». Прогнать на 200, 301, 404, 500, 999. Сделать двумя способами: `if/else if` и `switch(true)`.

## Критерии приёмки

* Корректно определяются все четыре категории и случай «неизвестный код».
* Реализованы оба варианта; понятно, чем `switch(true)` отличается от обычного `switch` по значению.
* В `switch` осознанно расставлены `break` (или объяснено, что даёт их отсутствие — fall-through).
